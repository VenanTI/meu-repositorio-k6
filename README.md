# Meu Repositório K6 — Portfólio de Estudo (Fictício)

Este repositório contém scripts de teste de performance e exemplos feitos com **k6** para fins de estudo. Trata-se de um portfólio de estudo e fictício — os dados e cenários são usados apenas para aprendizado.

Conteúdo principal:
- vários scripts `.js` para k6 (ex.: `maonamassa2.js`, `aula1.js`, `web1.js`)
- `dados.json` com dados de teste
- `usuarios.csv` e um arquivo HTML de exemplo

Executando um teste (exemplo):

```bash
# instale o k6: https://k6.io/docs/getting-started/installation/
k6 run maonamassa2.js
```

Observação: os scripts são exemplos educacionais. Use-os apenas em ambientes de teste controlados.
